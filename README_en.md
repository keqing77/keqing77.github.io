## Quick Start

### Install

```bash
# clone
git clone `repoUrl`

# install
pnpm install

# run
npm run start

# build
npm run build
```

### Structure:

```
├─ config
├─ public
├─ scripts
├─src

│   ├───pages
│   │   └───interview
│   └───theme
│       └───Footer
```
