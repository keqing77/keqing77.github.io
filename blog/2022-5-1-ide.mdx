## 编辑器美化

> 笔者为前端工程师, 主要使用的开发语言为 JavaScript, 因此使用 Webstorm 和 Viual Studio Code 较多, 偶尔写点 Java (Intellij IDEA) , Rust (Visual Studio Code)

### Visual Studio Code

#### 云同步

> 登录 github 账号并开启 Setting sync. 这样你的配置就会保存在云端, 每次安装都可以恢复之前的配置.

#### 配置文件

> Visual Studio Code 配置集中在 setting.json

```json
{
  "backgroundCover.imagePath": "d:\\OneDrive - FNC\\picturue\\background\\beauty.png",

  "editor.fontSize": 18,

  "editor.tokenColorCustomizations": {
```
