2022个人年度总结

December 12, 2022 · 15 min read
lavard
Front End Engineer @ Thoughworks
想让这个世界听听自己的意见

学业总结​

双非切图仔的自我救赎

嗨！我是 lavard，笔者来自广东某一本高校, 专业是计算机科学与技术(CS), 方向是前端开发,出生于广东非珠三角最贫穷的地区(广东省 GDP 倒数第一的市区), 通过 读书改变命运 这一想法从我懂事起就从未改变. 直到今日, 我仍然认为 教育 对一个人是最为重要的, 没有正确的教育引导, 社会一切秩序将不复存在.

选择 CS​

2018 年, 感觉参加完高考的我对专业方向感到迷茫, 似乎自己并没有一技之长, 但是在了解到社会形势以及亲友的推荐后, 我选择了计算机科学与技术(CS)专业。 大一大二都是塞满课表的通识课和计算机基础课，到了大三，我开始感到危机感，开始思考未来的方向， 最终决定选择 web 前端方向， 学习过程中我也从一个什么都不懂，什么都要问的菜鸟逐渐成长为一个程序猿。逐渐如何在互联网上收集信息， 解决问题， 回想那段日子， 这段时光只能用 难熬的青春 来形容吧！ 对未来充满憧憬但又伴随焦虑， 即便是现在这种感觉也依旧存在。

校招之旅​

正式学习前端是在大三下前的暑假，2 个多月的时间我只能从最基础的三件套学起，还没接触到框架。白天看视频学习，边敲代码边做笔记。累了就刷博客以及在牛客网刷面经 以及在 掘金 看总结文。随后返校开始准备秋招，校招期间我已经积累了一定的项目经验以及博客撰写， 但还存在唯一的短板就是算法，虽然前端实际业务中并无涉及过多算法，但算法依然是考察一个程序员逻辑思维的重要途经(其实就是内卷加剧)，所以每天都在坚持刷个几道算法题，不敢说这能使我算法水平提升显著，但对手撕算法的恐惧感确实在逐渐下降。甚至不少笔试都能全部 AC，这给我带来很大的信心。在面试求职的过程中我也在不断总结优劣， 不断改进，因为研发岗的笔试基本都是在牛客或者赛码网（少数使用自己的平台）写题，题型主要是选择题和算法题，时长普遍在一到两个小时。在感觉不满意的情况下，便会刻意在这些平台上练习。而面试无非就是八股文~ 感受就是只要 基础功扎实，便可成为 offer收割机

顺利入职​

首先我要祝贺自己，在 2022 年 6 月，顺利结束了大学四年本科生涯，并在今年 9 月份，在 Thoughworks 开启了前端工程师的旅程。

生活总结​

友情​

本科毕业，我与众多好友迎来了分别，他们的大多数和我一样进入企业开始自己的职业生涯，有的保研或考研至其他学校，这一别不知还要多久才能再次相见，江湖路远，期待我们都有光明的前途。

亲情​

这一年，很明显能够看出父母亲又年老了。我很惭愧，对我的家庭非常惭愧，已经二十多岁的人了，还没有成为家里的顶梁柱，父亲还是需要日夜班两班倒养家，母亲也因长年操劳很早就患了糖尿病。

对我自己说，很抱歉，还没有成为自己想要自己的成为的样子， 有些是自己选择但更多是被迫选择， 但正如 短门先生说 生活是旷野，不是轨道，我有权选择自己的生活 ， 所以挣扎着活下去吧！

技术总结​

开始写 Rust​

22 年因为某些机缘巧合了解到 Rust 并且开始想要学习，但尝试了好几次都没有坚持下去，因为太难了。 不止一次被困难挫败，之所以学习 Rust 而不是其他语言，是因为学习 Rust 能让我更加从底层知识去了解整个计算机体系，而不只是 API 调用工。

Rust 事实上已经成为我最喜欢的编程语言了，Rust 的性能和表达能力都很优秀，就业岗位暂时不是很多意味着还没有很卷，大企业和 linux 对 rust 的关注代表着它的前途，希望身边还没有开始 Rust 的同学马上开始行动。（我就是 Rust 布道者 , 虽然平时我还是在写js）

参与开源项目​

参加了今年的“开源软件供应链点亮计划”，体验不错。项目用的 go，需求比较简单，主要代码一两个晚上搞定，然后根据社区导师的建议小修小补。感谢社区导师@xuanwo, 给我提供了不少帮助。推荐学弟学妹们参加这个活动，有钱！中等难度项目 9000 块，赚的非常轻松。

在某群看到了 SOFAStack 社区发布的小任务，就给他们的 Layotto 项目做了两个 PR，顺便学习了 wasm 相关知识，特别是使用 wasm 写插件系统时，能力引入(导入表)和变长参数传递相关内容有了更加深入的了解。拿到了证书和马克杯，感谢。

写 copy-translator 项目时，给 egui 库做了几个小贡献，没想到头像居然出现在前 10 了，还被同样在学习 rust 的好朋友发现了，真不错。

今年总共有 3 个项目曾进入过 GitHub Trending，分别是 Golang 写的 proxypool（一个已经弃坑的项目，没想到还这么受欢迎）、copy-translator、good-mitm，后两个都是 rust 写的。

GitHub Followers 数目在 2021 年突破 1k。

参与开源的感觉真心不错，22 年再接再励！

内卷的加重​

这部分看过我《干！有人在卷我》这篇文章的同学能够看懂。

还是那个任课老师，听同学说，他今年在课上，说起字数的问题，“去年没有说字数的问题，很多同学问字数的问题，今年强调一下，我们没有字数限制。。。。”

然后，我那那篇文章，实验室一同学在某节课讨论时，被不小心传播开了。哈哈，听说他们今年已经开始 1w 起步了，身边有人已经写 1w7 了，这就是内卷的加重啊

然后，学校旁边的羊汤店，还在卷，饼可以免费不限量吃了。

行业相关​

平时都是比较关注计算机、互联网等领域的资讯, 从中可以了解技术风向和发展趋势以及就业形势等等, 就是近几年，已经看到不少互联网公司大幅裁员的新闻，其中最为恶劣还有应届生裁员等等, 互联网"寒冬"也许正在到来, 但不管如何, 依旧会选择在这行业挣扎. 当然也不是没有想过赚够留学的钱然后润(run)去欧州读 PhD, 但终究是想象不敢付诸于行动, 毕竟国内的才是我最熟悉的.

同时，今年我还见证了许多历史事件,俄乌冲突,上海疫情... 这些事件都充分证明了 人只能被形势所推, 而强者造就形势.

经济下滑​

互联网已经从增长时代 逐渐过渡 到 存量时代, 互联网大面积的裁员便是佐证. 看不过来的撕三方裁应届.

  • B 站、京东 毕业,率先拉起裁员浪潮
  • 杭州 有赞 裁员全部校招应届生
  • 深圳 转转 5 月裁员全部校招生
  • 杭州 税友 6 月份毁约应届生
  • 北京 北森6 月毁约应届生
  • ....

讨论的消亡​

我一直对中文互联网社区有强烈不满， 但是我也不确定这种不满的来源。 但可以肯定的是， 中文社区的讨论正在消亡。B 站、知乎、微博等等平台的评论限制审查，不符合氛围的评论直接屏蔽，留下的只是想让我们看到的，讨论区评论区都只是 情绪输出而不是 观点输出 ， 这真是令我失望至极，但又无力改变。

其次是技术社区的堕落，中文技术论坛像 CSDN 抄袭灌水而不加管理的状态以及持续不知多久，长期导致的结构就是中文搜索的质量非常低下，即便使用Google，首页也时常出现以上情况，更有甚者，国内厂商以及某些利益熏心的开发者更是自建内容农场，爬取他人网页聚合，既侵犯他人版权又导致想要真正查询这篇文章的用户浪费时间，这种情况下，技术社区的讨论也会消亡。

顺便推荐一篇文章：中文互联网中“讨论”的消亡

GitHub Copilot​

GitHub 用其托管的开源代码训练的人工智能插件，能够有效协助开发者写代码，特别是重复性的代码段, 一个 tab 就完成工作的感觉真的很爽,拿来刷力扣相当于面向答案编程了.

但是当我编写项目的时候，它的自动补全不断的自以为是打断我的思路，不仅没有提升我的效率,反而成为了我的累赘，因此在这个时候我会选择关闭它

区块链​

周围的人都在沉迷虚拟货币、挖矿、区块链等，时代的变化就是这么突然，

我不看好，但是不介意在这个没有监管的地方捞一笔，没有人和钱过不去。

反复的疫情​

2022 年，依旧笼罩新冠疫情的阴影下, 这给平时学习和工作的生活都带来许多不便利, 但生命无价, 希望疫情能早日消散吧!

思考​

这一年，因为看到、听到、经历到的各种事情，我也开始对人生、对理想、对未来、对社会、对人进行了更加深入的思考。这些思考还未形成文字，很多只是心里的一个想法，还需要后面的人生经历进行巩固和验证。

有几点比较自私的想法，我想记录在 2022 年，供未来的我思考和批判

  1. 以改变阶级层次为最终目标
  2. 作为程序员要努力活得久
  3. 家庭幸福、活得开心最重要
