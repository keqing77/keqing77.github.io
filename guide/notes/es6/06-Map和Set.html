# Set

Set

Set对象是值的集合，Set对象允许你存储任何类型的唯一值，无论是原始值或者是对象引用。Set中的元素只会出现一次，即 Set 中的元素是唯一的。

Set的具体实现可以参考05-集合

# Set属性和方法

// 返回Set的长度
- Set.prototype.size

// 往集合里面添加元素
- Set.prototype.add()

// 清空集合
- Set.prototype.clear()

// 删除某个元素(根据value)
- Set.prototype.delete()

// 判断Set中是否包含某个值
- Set.prototype.has()

// 遍历集合的每个元素
- Set.prototype.entries()
- Set.prototype.forEach()

// 分别获取集合中的所有键 和 值 ,实际上是一样的,因为在Set中 key = value
- Set.prototype.values()
- Set.prototype.keys() 

const arr1 = [6,6,6,6,6,6,6];

// 利用集合进行数组去重
const arr2 = [...new Set(arr1)];
console.log(arr2); // [ 6]

# Map

Map

又叫字典，Map 对象保存键值对，并且能够记住键的原始插入顺序。任何值(对象或者原始值) 都可以作为一个键或一个值

Map的具体实现可以参考06-字典

# Map属性和方法

// 返回Map的长度
- Map.prototype.size

// 清空Map
- Map.prototype.clear()

// 移除Map 对象中指定的元素(根据key)
- Map.prototype.delete(key)

// 遍历和迭代Map的每个元素
- Map.prototype.entries()
- Map.prototype.forEach(callbackFn, thisArg)

// 读取值
- Map.prototype.get(key)

// 添加键
- Map.prototype.set()

// 判断Map中是否包含某个键
- Map.prototype.has()

// 返回Map的所有键 值
- Map.prototype.keys()
- Map.prototype.values()

Map对象间可以进行合并，但是会保持键的唯一性

let first = new Map([
  [1, 'one'],
  [2, 'two'],
  [3, 'three'],
]);

let second = new Map([
  [1, 'uno'],
  [2, 'dos']
]);

// 合并两个Map对象时，如果有重复的键值，则后面的会覆盖前面的。
// 展开运算符本质上是将Map对象转换成数组。
let merged = new Map([...first, ...second]);

console.log(merged.get(1)); // uno
console.log(merged.get(2)); // dos
console.log(merged.get(3)); // three

# Map和Set的区别

  • 集合、字典 可以储存不重复的值，都可以去重！
  • 集合Set 是以 [value, value]的形式储存元素，字典Map 是以 [key, value] 的形式储存

# Map和Object的区别

  • Map的key可以是任意值，包括函数、对象或任意基本类型的,而Object只能是字符串类型
  • Map中的key是有序的，而Object的key是无序的;当迭代的时候，一个 Map 对象以插入的顺序返回键值。
  • Map 的键值对个数可以通过size获取，Object 的键值对个数只能手动计算
  • Map在频繁增删键值对的场景下表现更好

# WeakMap和WeakSet

WeakMap和WeakSet

除了Set和Map,ES6还增加了它们的弱化版本WeakMap和WeakSet。它们仅有的区别是

  • WeakMap类和WeakSet类没有 entries()、forEach()、keys()、values()方法。
  • 只能用对象作为键，而不能用其他类型的值作为键。

WeakMap和WeakSet的优点

  • 创建和使用这两个类主要是为了性能,它们是弱化的(用对象作为键),没有强引用的键.这使得JavaScript的垃圾回收器可以从中清除整个入口,不会保留对象的引用
  • 还有1个优点是必须用键才可以取出值.这些类没有entries()、forEach()、keys()、values()迭代器方法,除非你知道键,否则没有办法取出值.










 
 
 

const weakMap = new WeakMap();

const ob1 = {name: 'Tom'};
const ob2 = {name: 'Jack'};
const ob3 = {name: 'Lucy'};

weakMap.set(ob1, 'badboy@email.com');
weakMap.set(ob2, 'youngman@qq.com');
weakMap.set(ob3, 'prettygirl@gmail.com');

console.log(weakMap.has(ob1)); //true
console.log(weakMap.get(ob1)); // badboy@email.com
weakMap.delete(ob2);
