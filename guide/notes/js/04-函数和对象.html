# 函数

函数就是完成某个功能的一组语句，函数由关键字 function 函数名加一组参数定义

函数在定义后可以被重复调用，通常将常用的功能写成一个函数

1.函数声明

function sum1(n1,n2)
{
    console.log(n1+n2);
}
sum1(1,2);  //3

# 闭包

闭包就是一个函数能够访问其函数外部作用域中的变量。 简单来说闭包就是定义在函数体内部的函数。

闭包的三大特点为：
1、函数嵌套函数
2、内部函数可以访问外部函数的变量
3、参数和变量不会被回收。

# 立即执行函数(IIFE )

什么是立即执行函数，和闭包是一样的吗？


在Javascript中，一对圆括号“()”是一种运算符，跟在函数名之后，表示调用该函数。 比如，print()就表示调用print函数。

有时，我们需要在定义函数之后，立即调用该函数。 这时，你不能在函数的定义之后加上圆括号，这会产生语法错误。 产生这个错误的原因是，Javascript引擎看到function关键字之后， 认为后面跟的是函数定义语句，不应该以圆括号结尾。

解决方法就是让引擎知道，圆括号前面的部分不是函数定义语句，而是一个表达式， 可以对此进行运算。你可以这样写：

(function(){ /* code */ }()); 

// 或者

(function(){ /* code */ })();

这两种写法都是以圆括号开头，引擎就会认为后面跟的是一个表示式，而不是函数定义。
所以就避免了错误，这就叫做“立即调用的函数表达式”


以下的写法都可以让我们自动调用函数：
var i = function(){ return 10; }();

true && function(){ console.log('hello world'); }();

0, function(){ console.log('hello world'); }();

!function(){ console.log('hello world'); }();

~function(){ console.log('hello world'); }();

-function(){ console.log('hello world'); }();

+function(){ console.log('hello world'); }();

new function(){ console.log('hello world'); }

// 只有传递参数时，才需要最后那个圆括号。
new function(){ console.log('hello world'); }()

# 对象

JavaScript并不是严格的面向对象语言

JavaScript没有类的概念,JavaScript是通过原型链继承的方式的

对象是{}关键词。只要看到{}，那么它就一定是一个对象。

一个对象可以拥有属性(property)和方法(method)，

  • 属性是键值对的形式, 键和值通过:分割，比如下例，type是属性名(key)，属性值(vakue)是'哈巴狗'
  • 方法以函数定义被存储在属性中。
  • this关键词 this指的是它所属的对象 
dog {
    type:'哈巴狗',
    bark : function() {
    return '我的类型是' + " " + this.type;
  }
}

Object.create(null)与{}创建对象的区别


其实就是考察Object.create()的原理,Object.create()是将原型指向()里面的对象，指向null则代表它是一个完全干净的对象，而{}创建对象实际上是构造函数Object去创建的，原型是Object。这就是两者创建对象的区别。
