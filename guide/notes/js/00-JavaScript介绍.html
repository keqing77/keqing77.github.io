# JavaScript介绍

JavaScript前景

根据2021年Stack Overflow年度报告，JavaScript连续九年成为最常用的编程语言。对于大多数开发者来说，编程就是web编程。

# 什么是JavaScript?

JavaScript是一门单线程弱类型脚本语言

不像后端语言五花八门(Java GO PHP C# Rust)，前端唯一可选择的编程语言只有JavaScript

# JavaScript非常奇葩

小心JS的类型转换!

弱类型就是类型是不固定的，会随着运行而进行隐式类型转换，在JS中字符串可以用单引号''也可以用双引号""，它们没有区别。在项目迭代维护过程中，JS这个特性常常让人捉摸不透，因此学习给JS加了类型规范的TypeScript是很有必要的。但我相信，即便如此，你依然会爱上JavaScript的。
