# setup

setup函数

setup函数是 Vue3 中新增的函数，它是我们在编写组件时，使用 Composition API 的入口

  • 写法一: setup()
  • 写法二: <script setup> </script>
  • setup函数是处于 生命周期函数 beforeCreate 和 Created 两个钩子函数之间的函数，此时组件的 data 和 methods 还没有初始化，因此在 setup 中是不能使用 this 的,所以 Vue 为了避免我们错误的使用，它直接将 setup 函数中的 this 修改成了undefined
  • setup函数只能同步使用，不能用async将其设为异步
  • setup 函数接收两个参数 props和 context， 语法为：setup(props,context){}

props

props 里面包含父组件传递给子组件的所有数据。在子组件中使用 props 进行接收 props 是响应式的， 当传入新的 props 时，会及时被更新 由于是响应式的， 所以不可以使用 ES6 解构，解构会消除它的响应式

context

context 里面包含 attrs, slots, emit 等数据方法 attrs：获取组件上的属性 slots：获取 slot 插槽的节点 emit ：emit 方法（子组件向父组件传递数据）

# 新特性元素

import {
	reactive, // 声明对象变量 (响应式)
	readonly, // 声明对象变量 (响应式 / 只读, 修改会报错)
	ref, // 声明与ref相关变量 (响应式; 非模板通过 .value 访问值)

	shallowReactive, // 只有第一层对象可 响应式
	shallowReadonly, // 只有第一层对象可 响应式 / 只读
	shallowRef, // 改变值后变为 非响应式
	toRaw, // 获取 非响应式对象
	markRaw, // 转为 非响应式对象

	isReactive, // 判断是否 reactive 声明的
	isReadonly, // 判断是否 readonly 声明的
	isProxy, // 判断是否 reactive / readonly 声明的
	isRef, // 判断是否为 ref 声明的

	onBeforeMount, // beforeMount, 渲染初始化前
	onMounted, // mounted, 渲染初始化后
	onBeforeUpdate, // beforeUpdate, 内容更新前
	onUpdated, // updated, 内容更新后
	onBeforeUnmount, // beforeDestroy, 组件退出前
	onUnmounted, // destroyed, 组件退出后
	onErrorCaptured, // errorCaptured, 异常

	watch, // 监听
	watchEffect, // 监听, 调用 setup 立即执行
	computed, // 计算

	provide, // 注入依赖
	inject, // 接收依赖

	onRenderTracked, // 调试钩子
	onRenderTriggered, // 调试钩子
} from 'vue'
