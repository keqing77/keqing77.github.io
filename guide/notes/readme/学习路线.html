# 大前端学习路线

该学习路线为个人学习情况(包括现在和未来)，参考阿里巴巴给出的前端学习路线图 (opens new window)和个人情况综合绘制出的

TIP

大前端学习路线思维导图

WARNING

学习前端六个阶段

# 第一阶段

前端开发人员就基础的能力就是网页布局和效果实现。也就是常说的切图仔。 这个阶段需要掌握的是HTML+CSS。HTML需要掌握常用标签，CSS则掌握Flex布局、定位和媒体查询等，如果掌握了以上技能，你可以实现90%以上的静态页面，刚开始可能会觉得标签多，但常用的就那么几个，属性也是同理，熟练之后都是非常简单的，检验自己的成果可以去TB买一些PSD图来还原实现，或者直接去苹果官网等F12看源码模仿实现，这个阶段HTML一天就大概掌握了，css花费时间预计1~2周，最后一周练习检验成果

  • 难度指数:⭐️
  • 花费时间:2~3周
  • 掌握内容:HTML常用标签、表单；CSS边距/定位/Flex布局和@media媒体查询
  • 检验成果:根据设计图/在线网页实现一套静态页面(Mobile/PC要自适应)

# 第二阶段

学完HTML+CSS后，网页是纯静态的，也就是用户和网站没有交互。这时候你就要学习JavaScript了。这个阶段是比较难的一部分，特别是考虑到JavaScript与其他编程语言的差异，但前端工程师说到底就是考察JavaScript能力，HTML与CSS占前端工程师的比重只有10%，学会JavaScript 才能说真正成为一名前端开发者。学这部分也不要太执着，非要达到精通的程度(实际上根本做不到)，才往下学习。因为以后学习阶段，还会不断的使用这些JS语法。

这个阶段预计要花费一个月时间，这个阶段要求掌握基本语法和DOM操作，能实现一些效果(tab标签页、轮播图、表单验证、) 每个阶段都只能打基础从掌握到熟练，即便从业十年，也鲜有人敢称自己为’精通‘的

  • 难度指数:⭐️⭐️⭐️
  • 花费时间:一个月
  • 掌握内容:JS变量和数据类型、函数和对象、this的指向作用域和作用域链、原型和原型链、事件循环、正则表达式的校验 基本的DOM操作(增删查改)、事件冒泡、捕获和委托、定时器控制
  • 检验成果:原生JS实现各种效果(轮播图、tab标签页切换、图片放大..等等)

如果上面都做完了话，非常建议学习使用sass/less等CSS预处理器，以及Bootstrap和Tailwind这两个CSS框架。

# 第三阶段

学习完JavaScript(ES5)后，可能有些视频或者文章会推荐你JQuery，这个是完全错误的。JQ已经过时了，有点规模的公司都不会用，估计还在使用的都是老项目了，即使要用学也只是一个下午的事情(看文档直接撸)。这个阶段重点是学习ES6及之后的新特性和巩固前面基础，多加练习。 如果这三个阶段都顺利走完了，到这时候你已经足够资格找一份前端实习生/初级开发的工作了，这时候你已经具备一定水平的编程能力和解决问题能力。但这不意味你完事了，很多东西你学了只是知道了理解了，但是你真正会了是当你用出去的时候，这时候建议多做题练习或者在实习积累经验。

  • 难度指数:⭐️⭐️⭐️
  • 花费时间:一个月
  • 掌握内容:ES6后新特性如let、解构赋值、模板字符串、拓展运算符、类和继承、set和map、promise、async/await、proxy、reflect等等
  • 检验成果:把之前ES5实现的用ES6实现，尽可能多地使用ES6语法，ES6非常之重要

# 第四阶段

学习Node，如果你只是打算做前端的话，只需要了解常用npm命令即可，因为现代前端框架都是基于Node的，如果你像学点后端那么也可以学习Node，并用它搭建一个小型服务器，配合你的前端页面进行增删查改等交互操作。

  • 难度指数:⭐️(单前端,npm)/⭐️⭐️⭐️⭐️(Node全栈)
  • 花费时间:1天/1周
  • 掌握内容:前端:npm常用命令 node: node数据类型、node核心模块、文件系统、EventEmitter、Stream和Buffer、链接数据库、编写接口 以及一门数据库语言，sql推荐mysql，nosql推荐mongoDB
  • 检验成果:node搭建服务器并实现增删查改操作数据库等接口功能

# 第五阶段

学习Vue/React框架，你需要学一个流行的前端框架。推荐Vue.js，在国内这是使用人数最多的框架，相较于React更易上手，所以建议先学习Vue.js。(不推荐Angular的原因是太重了，如果之前学的是后端可以尝试)。还有一个原因是学完Vue之后，你可以无缝学习微信小程序(语法和Vue类似)，做完项目后可以根据需要刷点面试题。

  • 难度指数:⭐️⭐️⭐️⭐️
  • 花费时间:1周学习，1~2个月完成项目(取决于项目规模)
  • 掌握内容:基本语法、列表渲染、组件和组件间传值、路由、状态管理、SPA/SSR、虚拟DOM等等
  • 检验成果:做个大项目

# 第六阶段

这个阶段你应该要用啥学啥或者想学啥就学啥，因为你已经有足够的本领支撑你不会饿死，但依然需要保持学习的习惯，要TypeScript？学！要Nuxt/Next?学！同理Flutter、Uniapp、ReactNative、electron等等，如果是打算学习Node，那么express和koa或者nest.js/egg.js 以及 一到多门数据库也是必须的(mysql、MongoDB、redis);当然还有其他后端语言及框架。建议这些东西在工作后学习，因为这些东西有的是选学（也就是根据公司的需求），建议优先学TypeScript，这个预计2周。还有一些其他东西没有详细说，诸如构建工具Webpack/Vite，版本控制git以及其他一些零散的知识点，这个更应该构建你自己的知识网络去学习。

  • 难度指数:⭐️⭐️⭐️⭐️⭐️
  • 花费时间:职业生涯
  • 掌握内容:能根据文档实现几个Demo
  • 检验成果:经得起推敲的商业性项目
