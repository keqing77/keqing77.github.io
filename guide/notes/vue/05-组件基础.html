# 组件

组件就是一个迷你的页面，它可以包含其他组件，也可以被包含在其他组件。比如当前页面，就是好几个组件组成的。

  • 导航栏
  • 侧边栏
  • 正文内容

当然这是分得很粗，实际上光导航栏就是好几个组件组成，如搜索框，导航条，logo等等

通过组件的组合，我们可以构建一个完整的页面。组件可以复用，你这个项目写了这个轮播图组件，下个项目也用得上就直接复制粘贴就完事了，
