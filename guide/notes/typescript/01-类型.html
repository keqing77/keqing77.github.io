# 基本类型

和JS保持一致

显式声明类型

使用冒号 : + 类型可以显示声明类型, 可在冒号后加空格










 




const str: string = '显式声明变量类型';
const num: number = 666;
const bool: boolean = false;
const u: undefined = undefined;
const n: null = null;
const sym: symbol = Symbol('symbol');
const bigInt: bigint = BigInt(123);


每次声明变量都要加类型, 不是很麻烦吗?实际上TS是具有类型自动推断的, 根据赋值的类型自动推导

typeof auto; // number
// 实际上等于  const auto: number = 123;

# TS拓展类型

# any类型

  • 任意值（Any）用来表示允许赋值为任意类型。
  • 声明一个变量为任意值之后，对它的任何操作，返回的内容的类型都是任意值
let arr: any[] = [11, "你好世界"]
1

# unknown类型

它主要用来描述类型并不确定的变量。

  • 与 any 不同的是，unknown 在类型上更安全。
  • 比如我们可以将任意类型的值赋值给 unknown，但 unknown 类型的值只能赋值给 unknown 或 any。

# never

never 表示永远不会发生值的类型。一个永远不会有返回值，或者产生死循环的函数，就可以加上never 几乎不用

// 只有产生报错才会产生错误返回
function fn1(str: string): never {
    throw Error(str);
}

// 只有条件为真才会产生值返回
function fn2(): never {
    while(true) {}
}
