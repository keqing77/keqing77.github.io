# Map

Map和Set的区别在与,Set的key和value是相等的,而Map不是

Map 又叫字典,或者映射,关联数组.

关联数组的意思是我们可以使用[]来访问对象的属性,将属性名作为'索引位置'即可,键和值之间有关联的关系! 在计算机科学中,字典经常用来保存对象的引用地址

# Map的实现

本文使用对象{}来实现Map

// 理想的清空下是用字符串作为键名,但JS是弱类型,不能保证,因此需要为构造函数实现一个toString方法
function defalutToString(item) {
  if (item === null) {
    return '_NULL'
  }
  if (item === undefined) {
    return '_UNDEFINED'
  } else if ( typeof item === 'string' || item instanceof String) {
      return `${item}`
  }
  return item.toString()
}

class Map {
  constructor(toStrFn = defalutToString) {
    this.tables = {}
    this.toStrFn = toStrFn
  }

  // 检测一个键是否在Map中
  hasKey(key) {
    return this.tables[this.toStrFn(key)] !== null
  }


// 因为我们将key转化为了字符串,但为了保存信息的需要,我们也需要保存原始的key
class ValuePair {
  constructor(key, value) {
    this.key = key
    this.value = value
  }
  toString() {
    return `[#${this.key}:${this.value}]`;
  }
}
  // 设置一个键值对
  Set(key,value) {
    if( key != null && value != null) {
      const tableKey = this.toStrFn(key)
      this.tables[tableKey] = new ValuePair(key, value)
      return true;
    }
    return false;
  }

  // 移除一个键值对
  remove(key) {
    if(this.hasKey(key)) {
      const tableKey = this.toStrFn(key)
      delete this.tables[tableKey]
      return true
    }
    return false;
  }  

  // 获取一个键的值
  get(key) {
    const valuePair = this.table[this.toStrFn(key)];
    return valuePair === null ? undefined : valuePair.value;
  }

  keyValues() {
    return Object.values(this.tables)
  }

  keys() {
    return this.keyValues().map(valuePair => valuePair.key);
  }

  values() {
    return this.keyValues().map(valuePair => valuePair.value);
  }

  forEach(callbackFn) {
    const valuePairs = this.keyValues()
    for(let i = 0; i<valuePairs.length; i++) {
      const returnValue = callbackFn(valuePairs[i].key, valuePairs[i].value)
      if(returnValue === false) {
        break;
      }
    }
  }

  size() {
    return Object.keys(this.tables).length
  }

  isEmpty() {
    return this.size() === 0
  }

  clear() {
    this.tables = {}
  }

  toString() {
    if(this.isEmpty()) {
      return '';
    }
    const valuePairs = this.keyValues()
    let objString = `${valuePairs[0].toString()}`
    for(let i = 1; i<valuePairs.length; i++) {
      objString = `${objString},${valuePairs[i].toString()}`
    }
    return objString;
  }

}

# HashMap 散列表

散列表叫做HashTable,也叫做HashMap,它是字典的一种实现,特点是通过散列函数去寻址

散列算法

散列算法的作用是尽可能快地在数据结构中找到一个值,在数据结构中获得1个值,大多数时候是需要迭代整个数据结构来找到它,而通过散列函数,能快速检索该值的具体位置.散列函数的作用是给定一个简直,然后返回值在表中的位置.
