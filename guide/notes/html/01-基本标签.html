# 功能标签

  • 图片标签 引入一张图片到页面，src属性的值要写图片的路径
    <img src="图片路径" alt="图片说明">
1
  • 链接标签 给页面文本或图片加点击跳转到另一个页面的功能
    <a href="http://www.baidu.com">百度一下，你就知道</a>
1

# 文本标签

包含普通文本

    <p>这是一块文本标签</p>

# 列表标签

文章列表等需要使用的标签,ul表示无序列表，ol表示有序列表，区别在于li前面是否有序号

    <ul>
        <li>文章标题a</li>
        <li>文章标题b</li>
        <li>文章标题c</li>
        <li>文章标题d</li>
        <li>文章标题e</li>
    </ul>

# 标题标签

    <h2>二号标题</h2>
    <h3>三号标题</h3>
    <h4>四号标题</h4>

# 文本格式化

还有一些标签可以直接给文本加样式

<br />  换行
<hr />  水平线
<b></b>	加粗
<i></i>	加斜体
<s></s>	删除线
<u></u>	下划线
<sup></sup>	内容上标
<sub></sub>	内容下标
<pre>保留文字在源代码中的格式,使得页面中显示的内容和源代码中的格式一致</pre>

# HTML实体

如果你打很多个空格，html都会解析成1个空格，如果你就需要多个空格，这时候你就需要HTML实体了。

&nbsp;	空格
&lt;	<
&gt;	>
&quot;	"
&copy;	版权符号C
&reg;	注册商标R
&amp;	&

此框架非彼框架，只是一个结构的描述。如Frameset和iframe等内嵌框架。 HTML框架的作用就是把浏览器窗口划分成多个子窗口。 每个子窗口可以载入自的HTML文档
