# 边距

外边距和内边距

margin 和 padding分别是外间距和内间距，margin是盒子与盒子之间的间隔，padding是内容的补白。 元素的背景属性也会作用于padding。

# 盒模型

页面渲染时，dom 元素所采用的 布局模型。可通过box-sizing进行设置，分为W3C盒子模型(标准盒模型)和IE盒子模型(怪异盒模型) 区别：

标准盒模型：width/height 只是元素内容content宽度，不包含 padding 和 border

IE盒子模型： width/height 既包含元素content宽度也包含了 padding 和 border

//设置标准模型
box-sizing: content-box;
//设置IE模型
box-sizing: border-box;
