# 1.3 @规则

@import		导入外部样式表及目标媒体,该规则必须在样式表头部最先声明
@charset	在外部样式表文件内使用。指定样式表使用的字符编码
@media		指定样式表规则用于指定的媒体类型和条件
@font-face	设置嵌入HTML文档的OpenType字体
@keyframes	指定动画名称和动画效果

# 1.5继承

继承就是是否沿用父级元素的样式，有些属性是继承属性，有些则不是。
inherit 关键字允许显式的声明继承性，它对继承和非继承属性都生效

继承属性一般是文本、列表、字体等属性。

# 1.6简写

margin: 0; 上下左右都为0
margin: 0 0;  上下  左右都为0
margin: 0 0 0； 上0 左右0 下0
margin: 0 0 0 0;  上0 右0 下0 左0

像这样可以简写的还有padding、background、border、flex等

# 1.7选择符

视频

十分钟上手 CSS Selector (选择器)

当有多个规则都能应用于同一个元素时，权重越高的样式将被优先采用。

  • !important:优先级最高(慎用)
  • STYLE：1000
  • #ID：100
  • .CLASS 10
  • ELEMENT 1

!important > 行内样式 > id > class = 伪类 > tag = 伪元素 这时候你可能有问题，该怎样覆盖 !important呢？ 只需要加上!important 再额外加上ID 或者类选择器来让它权重更高，但最好不要滥用!important，因为这会给后续的维护带来许多困难。

样式选择符

名称 描述
通配选择符 *
ID选择符 #id
类选择符 .class
tag选择符 body
群组选择符 body,p,span

关系选择符

名称 描述
包含选择符(所有被E元素包含的F元素) E F
子选择符(所有作为E元素的子元素F) E>F
相邻选择符(紧贴在E元素之后F元素) E+F
兄弟选择符(E元素所有兄弟元素F) E~F

伪类选择符

如:hover :focus等等，伪类以单冒号:开头，伪元素以::双冒号开头
