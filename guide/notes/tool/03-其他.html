# 终端篇

高颜值的命令行

在Linux/MacOS/WSL(windows Subsystem for Linux)下使用 on my zsh + spaceship主题 + powerline字体 打造完美终端体验

  1. 查看当前系统的 shell
echo $SHELL
1
  1. 查看系统安装的 shell
cat /etc/shells
1
  1. 安装 zsh
#linux/maco 安装
sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

# WSL（ubuntu） 安装
sudo apt-get install zsh 

#查看zsh版本，出来即成功
zsh --version

安装失败

如果这里遇到错误：curl: (7) Failed to connect to raw.githubusercontent.com port 443: Connection refused

那是因为github 被墙了。解决方法是手动将该脚本保存至本地目录，然后执行该脚本。或者使用gitee仓库的脚本。

sh -c "$(curl -fsSL https://gitee.com/shmhlsy/oh-my-zsh-install.sh/raw/master/install.sh)"
1
  1. 切换终端为zsh
chsh -s/bin/zsh
1
  1. zsh配置文件
vim ~/.zshrc
1
  1. 切换终端为zsh
source ~/.zshrc
1

# powerline字体

这个是要给系统安装字体的，过程很快，详情参考仓库地址powerline (opens new window)

  • 我编程字体喜欢用jet brain家的jetbrain Mono
  • 终端字体使用powerline，
  • MacOS下使用的软件是item2
  • Win下使用的是Windows Terminal(WSL)，
  • wsl的mnt目录就是你本机的磁盘目录，我在windows上的开发工作也几乎用WSL完成。吹爆WSL

# spaceship主题

  1. 克隆仓库
git clone https://github.com/spaceship-prompt/spaceship-prompt.git "$ZSH_CUSTOM/themes/spaceship-prompt" --depth=1
1
  1. 软连接目录到本地克隆仓库
ln -s "$ZSH_CUSTOM/themes/spaceship-prompt/spaceship.zsh-theme" "$ZSH_CUSTOM/themes/spaceship.zsh-theme"
1

3.配置zshrc文件

vim ~/.zshrc # 编辑zshrc文件
ZSH_THEME="spaceship"  #配置zsh主题为spaceship
1
