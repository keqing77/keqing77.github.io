# React-介绍


:::info React.js
React是由Facebook(现改名Meta)开源的一个用于构建用户界面的JavaScript库，也是目前全球使用人数最多，生态最丰富的js框架，目前最新版本是18,最大的亮点是加入了并发渲染器。
:::


## 为何学React?

> React 作为前端三大框架之一, 使用人数是最多的 , 背靠 Meta , 是现在最为流行的JavaScript库, 以npm下载量来说, React 是下载量是1000万, 而Vue 和Angular基本持平,保持在200万左右

## 学习React
> 学习框架只推荐官方文档

- [新官网beta](https://beta.reactjs.org/)
- [旧官网](https://reactjs.org/)

## 创建React应用

### CRA
```bash
# React不再支持全局安装create-react-app，官方建议使用npx来安装
npx create-react-app 项目名
cd 项目名
npm start
```

### Vite
```bash
pnpm create vite
#选择 react-ts
pnpm install
pnpm run dev
```
