## 基础语法

### 数据绑定
`<span>Message: {{ msg }}</span>` Vue使用双大括号的语法插值 , 轻松将` 组件实例的属性值` 关联在一起, 你还可以在其中写`JavaScript`表达式

```html
{{ number + 1 }}

{{ ok ? 'YES' : 'NO' }}

{{ message.split('').reverse().join('') }}

<div :id="`list-${id}`"></div>
```

### 属性绑定
`<img :src="xxx" />` 使用v-bind:属性名=值 ,进行属性值绑定, `v-bind`被称为`指令` , 你可以简写为 `:`

### 指令(directive)
> 指令是带有v-前缀的特殊属性 , 启发于` Angular` 的 `ng-xx`

#### v-if
```html
<script setup>
const seen = true;
</script>

<p v-if="seen">Now you see me</p>  // Now you see me
```

#### v-show
```html
<script setup>
const seen = true;
</script>

<p v-show="seen">Now you see me</p>  // Now you see me
```

#### v-model
```html
<script setup>
const searchText = true;
</script>

<input v-model="searchText" />
```

#### v-for
```html
<div v-for="item in items">
  {{ item.text }}
</div>

<div v-for="(item, index) in items"></div>
<div v-for="(value, key) in object"></div>
<div v-for="(value, name, index) in object"></div>

<div v-for="item in items" :key="item.id">
  {{ item.text }}
</div>
```
#### v-bind
```html
<!--
Note that there are some constraints to the argument expression,
as explained in the "Dynamic Argument Value Constraints" and "Dynamic Argument Syntax Constraints" sections below.
-->
<a v-bind:[attributeName]="url"> ... </a>

<!-- shorthand -->
<a :[attributeName]="url"> ... </a>
```
#### v-on
```html
<a v-on:[eventName]="doSomething"> ... </a>

<!-- shorthand -->
<a @[eventName]="doSomething">
```
#### v-slot
```html
<!-- Named slots -->
<BaseLayout>
  <template v-slot:header>
    Header content
  </template>

  <template v-slot:default>
    Default slot content
  </template>

  <template v-slot:footer>
    Footer content
  </template>
</BaseLayout>

<!-- Named slot that receives props -->
<InfiniteScroll>
  <template v-slot:item="slotProps">
    <div class="item">
      {{ slotProps.item.text }}
    </div>
  </template>
</InfiniteScroll>

<!-- Default slot that receive props, with destructuring -->
<Mouse v-slot="{ x, y }">
  Mouse position: {{ x }}, {{ y }}
</Mouse>
```
