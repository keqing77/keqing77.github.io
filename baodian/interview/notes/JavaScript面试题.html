# ==和===的区别

THEOREM

  • ==判断值是否相等，类型不同会进行隐式类型转换
  • ===判断值和类型是否相等，不会进行类型转换
  • 避免错误尽量使用===

THEOREM

# new操作符干了什么？

THEOREM

  1. 创建一个空的简单JavaScript对象（即{}）；
  2. 为步骤1新创建的对象添加属性，将该属性链接至构造函数的原型对象 ；proto

# 跨域，原因和解决

THEOREM

跨域是由于浏览器的同源策略

同源策略

  • 域名、协议、端口号要一致，否则ajax请求失败

为什么要有同源策略？

  • 只有浏览器才会存在跨域,服务器或者小程序这些是不存在跨域的。
  • 是出于安全考虑，没有同源策略限制，你网站的cookie就会被任意来源的ajax获取，泄露信息。

跨域解决方案

  • jsonp
  • cors
  • node.js正向代理
  • nginx反向代理
  • webpack/vite/axios 配置 proxy
  • websocket(本身不存在跨域限制)
  • iframe + windows.postmessage
  • iframe + windows.document.domain
  • 浏览器开启跨域，不推荐
